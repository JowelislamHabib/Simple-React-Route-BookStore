# 📚 Simple React BookStore - Personal Bookstore Navigator

[**🚀 Live Demo**](https://jowel.bintofajjal.com/project-reactjs/simple-bookstore/) &nbsp; ![Live Status](https://img.shields.io/badge/Live-Demo-brightgreen?style=flat-square&logo=browser)

Welcome to **Simple React BookStore**, a premium book management application that helps you discover, track, and organize your reading journey. Built with the latest modern web technologies, it offers a seamless and aesthetically pleasing experience for every book lover.

---

## 🛠️ Technology Stack

This project is built using a state-of-the-art tech stack for performance, scalability, and design:

- **Frontend Core:** [React 19](https://react.dev/)
- **Routing:** [React Router 7](https://reactrouter.com/)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/) & [DaisyUI v5](https://daisyui.com/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Notifications:** [React Toastify](https://fkhadra.github.io/react-toastify/introduction/)
- **State Management:** React Hooks & Local Storage
- **Build Tool:** [Vite 8](https://vite.dev/)

---

## ✨ Key Features

### 1. Dynamic Book Discovery
Explore a curated collection of books with high-quality thumbnails and key metadata. Click on any book to dive deep into its summary, author details, genre tags, and rating, all presented in a beautifully structured layout.

### 2. Smart Reading Lists
Keep track of your literary adventures with dual-list management. Easily add books to your **"Read List"** or **"Wishlist"**. The application intelligently prevents duplicates and provides instant feedback via toast notifications.

### 3. Persistent User Experience
Never lose your progress! Your reading lists are automatically synchronized with your browser's **Local Storage**, ensuring your data persists even after refreshing the page or closing the browser.

## 🚀 Getting Started

To run this project locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/JowelislamHabib/Simple-React-Route-BookStore.git
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Run the development server:**
   ```bash
   npm run dev
   ```
4. **Open in browser:**
   Navigate to `http://localhost:5173`
